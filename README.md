# Shabnam Enterprises

A landing page for Shabnam Enterprises, an industrial contracting and workforce solutions company established in 1999.

## Project Overview

- **Home page:** `index.html`
- **Built with:** HTML, Tailwind CSS, Font Awesome
- **Highlights:** modern hero section, service portfolio, client brands, and contact CTA.

## How to use

1. Open `index.html` in a web browser.
2. Replace placeholder images and contact details with real company assets.
3. Deploy to any static hosting service such as GitHub Pages, Netlify, or Vercel.

## Notes

- The page loads Tailwind CSS from CDN.
- Replace the demo Cloudinary images with actual brand or product imagery as needed.
